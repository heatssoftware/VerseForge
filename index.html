Frequently Asked Questions

What is VerseForge?

VerseForge is a desktop app that lets you sync lyrics with music in real time, animate them visually, and fully customize the experience with themes, code, or visual editing.

Will it work with Spotify?

Yes! Integration is planned via desktop audio capture and Windows API for every windows supported streaming service.

Can I make my own themes?

Yes! A built-in editor lets you customize everything with visuals or code.

Is it free?

There will be a free version with access to all core features.
Premium users will get access to advanced tools, more customization options, cloud sync, and upcoming experimental features.
The free plan is fully usable, but upgrading unlocks the full potential of VerseForge.
